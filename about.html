A little about me...

My name is Brock Mowry and I am 19 year old programming student from Toronto CA. Growing up, I lived and breathed computers and, for as long as I can remember, there was always one in my room. My programming journey started back in 5th grade after discovering the MCP (Mod Coder Pack) for Minecraft. This "pack" allowed for alterations to the game's Java code. So, with the help of some YouTube tutorials, I picked up Java quickly.

This was only the start of a lifetime love for programming. Since then, I have developed quite the arsenal of programming knowledge and, with the help of my current program at Georgian, I hope to one day become an excellent Software Developer.
